My Projects
YelpCamp

YelpCamp is a web application that allows users to explore campgrounds. Users may view the information, photos, and reviews of each campground, as well as the profiles of the campground author and reviewers. Also, users can make their own campgrounds and post their own reviews. Made following Colt Steele's Udemy course "The Web Developer Bootcamp 2022", while adding my own features as well.

Visit YelpCamp Visit Git repo
A Bite of Home

A Bite of Home is a web application that helps home chefs to start their private businesses by allowing them a platform to sell their home-cooking, as well as post their recipes. Users can purchase home-cooked meals, as well as look up recipes from these home chefs. This is the second BCIT group project I've worked on.

Visit A Bite of Home Visit Git repo
InstaPlan

InstaPlan is a web application that helps to provide information about attractions, restaurants, and 2030 Olympic games (hypothetically) in Vancouver. This application also helps generate an itinerary for busy 2030 Vancouver Olympic tourists. This is the first BCIT group project I've worked on.
